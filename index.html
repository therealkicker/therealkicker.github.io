The Lamp

Our specially designed lamp is WiFi-controlled and reacts sensitively on movements like goals or strong kicks during the game.

The Real Kicker

This is how it works: a special light barrier recognizes and tracks when a goal is scored. As soon as the ball passes the goal line, it sends a signal via WiFi.

The Web App

Additional to the Real Kicker you can use the web app. Set up a new match before you begin a game and it counts all goals automatically. See who's the best player in the team ranking!
